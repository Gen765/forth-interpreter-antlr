Joc de proves per a l'intèrpret mini Forth

>>> from forth import interpret

1. EXEMPLES DEL README

>>> interpret('1 2 3 .s')
[1, 2, 3]

>>> interpret('5 2 .s - .s .')
[5, 2]
[3]
3

>>> interpret(': min 2dup < if drop else swap drop endif ; 3 2 min .')
2

>>> interpret(': signe dup 0 < if drop -1 else 0 > if 1 else 0 endif endif ; -2 signe . 0 signe . 5 signe .')
-1
0
1

>>> interpret(': faux dup rot * swap 1 - dup 2 < if drop else recurse endif ; : factorial dup 2 < if drop 1 else dup 1 - faux endif ; 3 factorial . 4 factorial . 5 factorial .')
6
24
120

2. OPERACIONS DE PILA

>>> interpret('1 2 3 rot swap .s')
[2, 1, 3]

>>> interpret('1 2 3 4 2swap 2over .s')
[3, 4, 1, 2, 3, 4]

>>> interpret('3 4 2dup 2drop .s')
[3, 4]

3. OPERACIONS COMBINADES

>>> interpret('3 4 over over * rot rot + .s')
[12, 7]

>>> interpret('10 3 / 10 3 mod .s')
[3, 1]

>>> interpret('5 dup * dup + .')
50

>>> interpret('5 5 = . 3 5 <> . 3 5 < . 5 3 >= .')
-1
-1
-1
-1

>>> interpret('1 2 < 3 4 > and . 1 2 < 3 4 < or . 0 not .')
0
-1
-1

4. PROCEDIMENTS

>>> interpret(': abs dup 0 < if 0 swap - endif ; -3 abs . 5 abs .')
3
5

>>> interpret(': max 2dup > if drop else swap drop endif ; 3 7 max . 9 2 max .')
7
9

>>> interpret(': nada ; 1 2 nada .s')
[1, 2]

>>> interpret('1 2 ( això és un comentari ) + .')
3

5. ERRORS

>>> interpret('1 . .')
1
Error: pila buida!

>>> interpret('5 0 /')
Error: divisió per zero!

>>> interpret('5 0 mod')
Error: divisió per zero!

>>> interpret('foo')
Error: procediment 'foo' no definit!

6. RECURSIVITAT

>>> interpret(': fib dup 2 < if else dup 1 - recurse swap 2 - fib + endif ; 1 fib . 4 fib . 10 fib .')
1
3
55

>>> interpret(': gcd dup 0 = if drop else swap over mod recurse endif ; 48 18 gcd . 100 35 gcd .')
6
5

>>> interpret(': pow2 dup 0 = if drop 1 else 1 - recurse 2 * endif ; 10 pow2 .')
1024

>>> interpret(': sum dup 0 = if else dup 1 - recurse + endif ; 10 sum .')
55

7. REDEFINICIÓ I COMPOSICIÓ

>>> interpret(': dup drop ; 1 2 dup .s')
[1]

>>> interpret(': + - ; 10 3 + .')
7

>>> interpret(': doble 2 * ; : quadruple doble doble ; 5 quadruple .')
20
